== Description ==

Last Wordpress Seo Plugin:Auto add keywords and description in the head meta of home,single,page,category and tag,also can custom title keywords and  description. 
Languages:English, Chinese simplified(zh_CN).
Last SEO option after installation. If you translate it, please notice me.

== Installation ==

1. Upload the folder Last Wordpress Seo Plugin to the `/wp-content/plugins/
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > Last SEO option,the illustration is available there.
